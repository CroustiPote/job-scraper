📦 Ton script est prêt

Télécharge job_scraper.py et suis le guide d'installation pour l'automatiser.

Et ensuite ?

🐍

Installe Python 3.11

Télécharge Python sur python.org — coche "Add to PATH" pendant l'install.

📦

Installe les dépendances

Dans le Terminal (Mac) ou PowerShell (Windows), lance :

pip3 install python-jobspy requests beautifulsoup4 lxml feedparser

▶️

Lance le script

Place job_scraper.py dans un dossier, puis dans le terminal :

python3.11 job_scraper.py

Un fichier offres_emploi.csv sera créé avec toutes les offres.

⏰

Automatise avec GitHub Actions

Pour que le script tourne tout seul chaque matin, crée un compte gratuit sur github.com et contacte la personne qui t'a partagé cet outil pour le guide complet.
